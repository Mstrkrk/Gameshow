How to run it
One item per line: Question | Answer | Points. The answer and points are optional — Question | Answer uses the default points, and a trailing number is always read as points.

The board starts blank. N walks the show: reveal the question + arm buzzers → N shows the answer → N moves to the next question + arms again. First contestant to hit 1 or 2 locks the other out; judge with R (full value) or W (minus half). If no one buzzes within 10 seconds the buzzers shut off automatically. Space re-arms the same question (steal), P steps back. After the last answer, N shows the final scores.

The 1/2 keys also match standard USB game-show buzzers, so you can add physical buttons later with no changes.

Sound cues (bottom row): Z walk-in theme · X "Garden Showdown" title · C welcome contestants · V walk-off music · B rimshot. Z and V are music (press again to stop); the rest are one-shots. The phone-QR panel is now Q. Drop your clips in assets/ as title.mp3, welcome.mp3, walkoff.mp3, and rimshot.mp3 (a built-in synth rimshot plays until the file is added).
