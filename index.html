How to run it
Put one question per line. Add | 200 at the end of a line to override that question's points; otherwise it uses the default. Step through with N (next) and P (previous).

Read the question, press Space to go live. First contestant to hit 1 or 2 locks the other out. Judge with R (adds the full value) or W (subtracts half). Space re-arms the same question for a steal; N moves on.

The 1/2 keys also match standard USB game-show buzzers, so you can add physical buttons later with no changes. Keep bumper music on a separate device through the PA.
