How to run it
Put one question per line. Add | 200 at the end of a line to override that question's points; otherwise it uses the default.

The board starts blank. Press N to reveal the next question and arm the buzzers in one step. First contestant to hit 1 or 2 locks the other out; judge with R (adds the full value) or W (subtracts half). Then N again for the next one. Space re-arms the same question (for a steal), and P steps back.

The 1/2 keys also match standard USB game-show buzzers, so you can add physical buttons later with no changes. Keep bumper music on a separate device through the PA.
